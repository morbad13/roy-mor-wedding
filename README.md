`https://morbad13.github.io/roy-mor-wedding/`

Assets:

- `BANK_NAME`
- `BRANCH_NUMBER`
- `ACCOUNT_NUMBER`
- `ACCOUNT_HOLDER`
- `WHATSAPP_PHONE`
- `WEDDING_DATE`
- `VENUE_NAME`
